# Local Document Chat

## 📌 Project Overview

Local Document Chat is an AI-powered document assistant that enables users to upload PDF documents and ask questions in natural language. The system retrieves relevant information from the uploaded documents using Retrieval-Augmented Generation (RAG) and generates accurate responses with the help of the Gemini API.

## 🚀 Tech Stack

- Python
- SQLite3
- LangChain
- ChromaDB
- Gemini API
